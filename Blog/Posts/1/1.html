How to change System Font in Windows 10? and How to reset Windows System default Font?

How to change System Font in Windows 10? and How to reset Windows System default Font?





Hello Friend,

Today I must Demonstrate, How to Change System font in windows 10? I have executed in this Post as step by step.

So, How to Change System Font?

Step 1 :- Go to the Notepad.

Step 2 :-Scroll Down and Copy Font Changer Coad.

Step 3 :-After Copying , past in your notepad and Enter Font of you Want to set for your System, Then enter you Want Font name.

Step 4 :-Save this file for this name, Font Change.reg

Step 5 :- Run this file, and agree for all.

Step 6 :- Restart your Computer.

 

Now, How to reset Windows System default Font?

Step 1 :- Go to the Notepad.

Step 2 :-Scroll Down and Copy Windows Default System Font Coad.

Step 3 :-After Copying , past in your notepad.

Step 4 :-Save this file for this name, Default Font.reg

Step 5 :- Run this file, and agree for all.

Step 6 :- Restart your Computer.



This Coad for using in Font Change and reset.
